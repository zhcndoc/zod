## 介绍

Zod 是一个 TypeScript 优先的验证库。使用 Zod，你可以定义*模式*来验证数据，从简单的 `string` 到复杂的嵌套对象。

```ts
import * as z from "zod";

const User = z.object({
  name: z.string(),
});

// 一些不可信的数据...
const input = { /* 内容 */ };

// 解析结果是验证过且类型安全的！
const data = User.parse(input);

// 因此你可以放心使用 :)
console.log(data.name);
```

## 特性

- 零外部依赖
- 支持 Node.js 和所有现代浏览器
- 极简：核心包大小 2kb（压缩后）
- 不可变 API：方法返回新实例
- 简洁接口
- 支持 TypeScript 和普通 JS
- 内置 JSON Schema 转换
- 丰富的生态系统


## 安装


```sh
npm install zod
```

> Zod 也可以在 [jsr.io](https://jsr.io/@zod/zod) 上以 `@zod/zod` 的形式获取。

Zod 提供一个 MCP 服务器，供代理搜索 Zod 的文档。要将其添加到您的编辑器，请按照 [这些说明](https://share.inkeep.com/zod/mcp) 操作。Zod 还提供一个 [llms.txt](https://zod.dev/llms.txt) 文件。

## 要求

Zod 在 *TypeScript v5.5* 及更高版本中测试通过。较旧版本可能可用，但不正式支持。

### `"strict"`

你必须在 `tsconfig.json` 中启用 `strict` 模式。这是所有 TypeScript 项目的最佳实践。
  
```ts
// tsconfig.json
{
  // ...
  "compilerOptions": {
    // ...
    "strict": true
  }
}
```
